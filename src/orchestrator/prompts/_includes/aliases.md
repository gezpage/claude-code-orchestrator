## Path aliases

In the prose below, these short tokens stand in for absolute paths. Substitute them when reading, writing, or running commands:

- `$REPO_ROOT` → `{{ repo_root }}`
- `$RUN_FOLDER` → `{{ run_folder }}`
- `$DOCS_ROOT` → `{{ docs_root }}`

SIGNAL_JSON output below uses fully-expanded absolute paths — emit them verbatim.
